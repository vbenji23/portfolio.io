## Installation et Utilisation

Ce portfolio est un site web statique, ce qui rend son déploiement très simple.

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/votre-nom-utilisateur/mon-portfolio.git
    cd mon-portfolio
    ```
2.  **Ouvrir le fichier `index.html`** :
    Il suffit d'ouvrir le fichier `index.html` dans votre navigateur web préféré.

    ```bash
    # Sur la plupart des systèmes d'exploitation
    open index.html
    ```
